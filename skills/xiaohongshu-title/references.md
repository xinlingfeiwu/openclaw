# Reference: Strategic Assets

## 1. High-CTR Keyword Dictionary

| Type                    | Keywords                                           |
| :---------------------- | :------------------------------------------------- |
| **Emotion (情绪)**      | 救命, 哭死, 绝绝子, 离大谱, 谁懂啊, 真的香, 破防了 |
| **Action (指令)**       | 闭眼入, 码住, 建议收藏, 别乱买, 快冲, 抄作业, 听劝 |
| **Urgency (紧迫)**      | 最后一天, 错过拍大腿, 手慢无, 绝版, 限时           |
| **Exaggeration (夸张)** | 天花板, 焊在脸上, 妈生感, 换头, 封神               |

## 2. Structural Templates (Formulas)

1.  **[Emoji] + [Audience Call] + [Benefit]**
    - _e.g.:_ 📢学生党必看！宿舍收纳神器...
2.  **[Negative Warning] + [Better Alternative]**
    - _e.g.:_ 别买iPad了！这个更香...
3.  **[Suspense] + [Result]**
    - _e.g.:_ 坚持了3天，没想到结果...
4.  **[Numbers] + [Process] + [Authority]**
    - _e.g.:_ 3个步骤，HR手把手教你改简历

## 3. Compliance Blacklist (Strictly Forbidden)

- **Absolute Terms**: 第一, 顶级, 国家级, 最强, 完美.
- **Medical Claims**: 治愈, 根除, 100%有效, 药到病除.
- **Superstition**: 转运, 必过(unless contextual), 算命.
